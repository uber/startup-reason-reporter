Protocols

The following protocols are available globally.

  • UBApplicationStartupReasonReporterPriorRunInfoProtocol

    Defines a storage interface for the UBApplicationStartupReasonReporter. On launch, should contain information regarding the prior app run. When data is set thereafter, a call to persist should persist the data for the next app launch.
